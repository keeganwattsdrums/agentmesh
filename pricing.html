🚀 Quick Start

1. Get your API key:

curl http://cuddly-bears-punch.loca.lt/key/generate

2. Install the SDK:

pip install agentmesh

3. Start building:

from agentmesh import Agent

agent = Agent(
    name="my-bot",
    api_key="your-key",
    memory=True
)

agent.remember("Important context...")
memories = agent.recall("what was that?")
