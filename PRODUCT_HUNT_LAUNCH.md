## Tagline

## Description

**The Problem:**
AI agents forget everything when they restart. Context windows are too small. Building persistent memory is complex and expensive.

**The Solution:**
AgentMesh provides persistent memory, semantic recall, and context management for AI agents — with a single API call.

**Key Features:**
✅ **Persistent Memory** — Store and retrieve agent context across sessions
✅ **Semantic Search** — Find relevant memories with natural language
✅ **Context Optimization** — Automatically manage context window limits
✅ **One-Line Integration** — `pip install agentmesh` and start building
✅ **Zero Config** — Works out of the box, scales when you need it

**For Developers:**
- Python SDK with async support
- RESTful API with OpenAPI spec
- Webhook notifications for memory events
- Self-hosted or managed options

**Pricing:**
- **Starter:** $19/month (1,000 agents)
- **Pro:** $99/month (10,000 agents)
- **Enterprise:** Custom (unlimited)

## FAQ

**Q: How is this different from Pinecone?**
A: Pinecone is a vector database — you still build the memory layer. AgentMesh is complete memory infrastructure with context optimization, semantic search, and agent-specific APIs.

**Q: What languages are supported?**
A: Currently Python with JavaScript/TypeScript coming soon. REST API works with any language.

**Q: Can I self-host?**
A: Yes! Open source core with managed cloud option.

**Q: How does pricing work?**
