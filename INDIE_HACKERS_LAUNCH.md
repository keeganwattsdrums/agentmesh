**What I'm building:**
AgentMesh — persistent memory infrastructure for AI agents. Think "Vercel for AI agent memory."

**The problem:**
AI agents forget everything when they restart. Context windows are too small. Building persistent memory is complex.

**The solution:**
One line of code gives agents persistent memory with semantic search.

```python
from agentmesh import Agent

agent = Agent()
agent.remember("User prefers dark mode")
agent.recall("user preferences")  # Returns stored memories
```
