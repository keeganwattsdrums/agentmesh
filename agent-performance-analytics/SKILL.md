# Agent Performance Analytics

Real-time performance monitoring, metrics tracking, and optimization insights for AI agents. Track latency, token usage, success rates, and cost per operation with beautiful dashboards and actionable recommendations.

## Use Cases

- Monitor agent response times and identify bottlenecks
- Track token consumption patterns across different operations
- Analyze success/failure rates and error patterns
- Compare performance across different models and configurations
- Generate reports for stakeholders on agent efficiency
- Set up alerts for performance degradation
- Optimize agent configurations based on data

## Installation

```bash
# Copy skill to your OpenClaw skills directory
cp -r agent-performance-analytics ~/.openclaw/skills/

# Install dependencies
pip install -r ~/.openclaw/skills/agent-performance-analytics/requirements.txt
```

## Quick Start

```bash
# Start collecting metrics
openclaw skills run agent-performance-analytics --mode=collect

# View dashboard
openclaw skills run agent-performance-analytics --mode=dashboard

# Generate report
openclaw skills run agent-performance-analytics --mode=report --period=7d
```

## Configuration

Create `~/.openclaw/skills/agent-performance-analytics/config.yaml`:

```yaml
# Metrics Collection
collection:
  enabled: true
  sample_rate: 1.0  # 1.0 = collect all, 0.1 = 10% sampling
  retention_days: 30
  
# Storage
storage:
  backend: sqlite  # sqlite, postgresql, timescaledb
  sqlite_path: ~/.openclaw/skills/agent-performance-analytics/metrics.db
  
# Dashboard
dashboard:
  enabled: true
  port: 8080
  refresh_interval: 30
  
# Alerts
alerts:
  enabled: false
  webhook_url: ""
  thresholds:
    latency_p95_ms: 2000
    error_rate_percent: 5.0
    cost_per_hour_usd: 10.0
```

## Usage

### Collecting Metrics

The analytics collector automatically hooks into agent operations:

```python
from agent_performance_analytics import AnalyticsCollector

collector = AnalyticsCollector()

# Wrap your agent calls
with collector.track(operation="summarize", model="claude-3-opus"):
    result = agent.summarize(text)
```

### CLI Commands

```bash
# Start metrics collection daemon
agent-analytics collect --config=config.yaml

# View real-time dashboard
agent-analytics dashboard --port=8080

# Generate performance report
agent-analytics report --period=24h --format=html --output=report.html

# Compare models
agent-analytics compare --models="gpt-4,claude-3-opus,gemini-pro" --metric=latency

# Export data
agent-analytics export --start="2024-01-01" --end="2024-01-31" --format=csv
```

### Python API

```python
from agent_performance_analytics import Analytics, MetricQuery

# Initialize analytics
analytics = Analytics(config_path="config.yaml")

# Query metrics
query = MetricQuery()
    .time_range("24h")
    .operation("chat_completion")
    .model("claude-3-opus")

metrics = analytics.query(query)

# Get insights
insights = analytics.get_insights()
print(f"Average latency: {insights.latency.avg_ms}ms")
print(f"P95 latency: {insights.latency.p95_ms}ms")
print(f"Total cost: ${insights.cost.total:.2f}")

# Generate optimization recommendations
recommendations = analytics.recommend()
for rec in recommendations:
    print(f"[{rec.priority}] {rec.message}")
```

## Metrics Tracked

### Performance Metrics
- **Latency**: Response time in milliseconds (avg, p50, p95, p99)
- **Throughput**: Requests per minute/hour
- **Time to First Token (TTFT)**: For streaming responses
- **Tokens per Second**: Generation speed

### Cost Metrics
- **Total Cost**: Aggregated spend by period
- **Cost per Request**: Average cost per operation
- **Cost per 1K Tokens**: Efficiency metric
- **Cost by Model**: Breakdown by model/provider

### Quality Metrics
- **Success Rate**: Percentage of successful operations
- **Error Rate**: Failed operations percentage
- **Error Types**: Categorized failure reasons
- **Retry Rate**: How often retries are needed

### Token Metrics
- **Input Tokens**: Tokens sent to models
- **Output Tokens**: Tokens generated by models
- **Total Tokens**: Combined usage
- **Token Efficiency**: Output per dollar spent
