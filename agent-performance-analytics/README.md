## 🎯 What is this?

Agent Performance Analytics tracks your AI agents' performance in real-time. Monitor latency, token usage, success rates, and costs with beautiful dashboards and actionable recommendations.

## 🚀 Quick Start

```bash
# Install
pip install -r requirements.txt

# Configure
cp config.example.yaml config.yaml
# Edit config.yaml with your settings

# Start collecting metrics
python analytics.py collect

# View insights
python analytics.py insights
```
